# 🚀 Улучшения профиля пользователя - RANO Komfort

## ✅ Реализованные функции

### 1. 🔐 **Сохранённые адреса**
- Добавление нескольких адресов доставки (Дом, Работа, и т.д.)
- Установка адреса по умолчанию
- Редактирование и удаление адресов
- API endpoints:
  - `GET /api/addresses?userId={id}` - получить все адреса
  - `POST /api/addresses` - создать новый адрес
  - `PUT /api/addresses/[id]` - обновить адрес
  - `DELETE /api/addresses/[id]` - удалить адрес

**UI особенности:**
- На мобильной версии: красивая карточка с кнопкой удаления
- На десктопе: сетка адресов с визуальным выделением адреса по умолчанию
- Форма добавления адреса с валидацией

---

### 2. 🔑 **Изменение пароля**
- Отдельная вкладка для смены пароля
- Проверка текущего пароля
- История паролей (сохраняется в БД)
- Требование минимум 6 символов
- API endpoint:
  - `POST /api/change-password` - изменить пароль

**Процесс:**
1. Пользователь вводит текущий пароль
2. Система проверяет его корректность
3. Пользователь вводит новый пароль и подтверждение
4. Старый пароль сохраняется в истории
5. Новый пароль устанавливается

---

### 3. 📸 **Фото профиля (аватар)**
- Загрузка и сохранение аватара
- Отображение в карточке профиля
- API endpoint:
  - `GET /api/profile/avatar?userId={id}` - получить аватар
  - `PUT /api/profile/avatar` - обновить аватар

### 4. ⭐ **Уровень лояльности**
- Отслеживание уровня лояльности пользователя
- 4 уровня: Бронза, Серебро, Золото, Платина
- Накопление баллов (1 балл = 1 тенге потрачено)
- Отображение общей суммы потраченных денег
- Визуальное отображение с цветовой кодировкой

**Уровни:**
```
Бронза (Bronze)     - 0 ₸
Серебро (Silver)    - 50,000 ₸
Золото (Gold)       - 100,000 ₸
Платина (Platinum)  - 250,000 ₸
```

## 🗄️ **Изменения в БД (Prisma)**

### Новые модели:

#### User
```prisma
model User {
  id              Int
  name            String
  email           String @unique
  phone           String?
  password        String
  avatar          String?
  
  addresses       Address[]
  orders          Order[]
  loyaltyPoints   LoyaltyPoints?
  passwordHistory PasswordHistory[]
}
```

#### Address
```prisma
model Address {
  id          Int
  userId      Int
  label       String      // "Дом", "Работа"
  fullAddress String
  city        String
  street      String
  building    String
  apartment   String?
  isDefault   Boolean
}
```

#### PasswordHistory
```prisma
model PasswordHistory {
  id           Int
  userId       Int
  passwordHash String
  changedAt    DateTime
}
```

#### LoyaltyPoints
```prisma
model LoyaltyPoints {
  id         Int
  userId     Int @unique
  totalSpent Float
  points     Int       // 1 балл = 1 тенге
  tier       String    // "bronze", "silver", "gold", "platinum"
}
```

## 📝 **Примеры запросов**

### Получить профиль с лояльностью
```bash
GET /api/profile?userId=1
```

Ответ:
```json
{
  "id": 1,
  "name": "Айгерим Сейткали",
  "email": "aigerim@mail.ru",
  "phone": "+7 (777) 123-45-67",
  "avatar": "АС",
  "loyaltyPoints": {
    "totalSpent": 11540,
    "points": 1154,
    "tier": "silver"
  },
  "addresses": [...],
  "orders": [...]
}
```

### Добавить адрес
```bash
POST /api/addresses
Content-Type: application/json

{
  "userId": 1,
  "label": "Работа",
  "fullAddress": "Алматы, пр. Аль-Фараби 77",
  "city": "Алматы",
  "street": "пр. Аль-Фараби",
  "building": "77",
  "apartment": null,
  "isDefault": false
}
```

### Изменить пароль
```bash
POST /api/change-password
Content-Type: application/json

{
  "userId": 1,
  "currentPassword": "oldPassword123",
  "newPassword": "newPassword456"
}
```
