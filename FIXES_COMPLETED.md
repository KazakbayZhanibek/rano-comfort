## 📡 Новые API endpoints:

### `/api/wishlist` (GET/POST)
```
GET  /api/wishlist           - получить список избранных товаров пользователя
POST /api/wishlist           - добавить/удалить товар из избранного
   body: { productId: number }
   response: { added: boolean }
```

### `/api/change-password` (POST)
```
POST /api/change-password    - изменить пароль
  body: {
    currentPassword: string,
    newPassword: string,
    confirmPassword: string
  }
  response: { success: true }
```
